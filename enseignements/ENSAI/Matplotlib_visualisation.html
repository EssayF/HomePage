Plus de dtails dans la documentation sur l'API de matplotlib pour paramtrer la couleur , les markers , et le style des lignes . MatplotLib est compatible avec plusieurs standards de couleur :

  • sous forme d'une lettre : 'b' = blue (bleu), 'g' = green (vert), 'r' = red (rouge), 'c' = cyan (cyan), 'm' = magenta (magenta), 'y' = yellow (jaune), 'k' = black (noir), 'w' = white (blanc).
  • sous forme d'un nombre entre 0 et 1 entre quotes qui indique le niveau de gris : par exemple '0.70' ('1' = blanc, '0' = noir).
  • sous forme d'un nom : par exemple 'red'.
  • sous forme html avec les niveaux respectifs de rouge (R), vert (G) et bleu (B) : '#ffee00'. Voici un site pratique pour rcuprer une couleur en RGB hexadcimal.
  • sous forme d'un triplet de valeurs entre 0 et 1 avec les niveaux de R, G et B : (0.2, 0.9, 0.1).

Titre, légendes et labels¶

On peut galement amliorer notre graphique en inserant :

- des tiquettes (abscisses et ordonnes) : $xlabel(texte), ylabel(texte)$
- une lgende : $legend()$
- la grille : $grid()$
- titre du graphique : $title(texte) $
- un texte sur le graphique : $text(x,y,texte) $
- une grille en filligrane : $grid()$

Exemple :

In [11]:
x = np.linspace(0, 2, 10)

plt.plot(x, x, 'o-', label='linear')
plt.plot(x, x ** 2, 'x-', label='quadratic')

plt.legend(loc='best')
plt.title('Linear vs Quadratic progression')
plt.xlabel('Input')
plt.ylabel('Output');
plt.show()

Ticks et axes¶

3 mthodes cls :

  • xlim() : pour dlimiter l'tendue des valeurs de l'axe
  • xticks() : pour passer les graduations sur l'axe
  • xticklabels() : pour passer les labels

Pour l'axe des ordonnes c'est ylim, yticks, yticklabels.

Pour rcuprer les valeurs fixes :

  • plt.xlim() ou plt.get_xlim()
  • plt.xticks() ou plt.get_xticks()
  • plt.xticklabels() ou plt.get_xticklabels()

Pour fixer ces valeurs :

  • plt.xlim([start,end]) ou plt.set_xlim([start,end])
  • plt.xticks(my_ticks_list) ou plt.get_xticks(my_ticks_list)
  • plt.xticklabels(my_labels_list) ou plt.get_xticklabels(my_labels_list)

Si vous voulez customiser les axes de plusieurs sous graphiques, passez par une instance de axis et non subplot.

Figures et subplots¶

Laffichage avec plt.show() ouvrira une fentre pour chaque figure cre. Plus intressant, on peut dans une mme figure, crer plusieurs graphiques. On utilise pour cela la commande subplot sous la forme subplot(p,q,n) o (p,q) reprsente la taille du tableau des sous-graphiques et n la position dans ce tableau : si p = 2 et q = 3, alors on cre une grille de 6 graphiques (2 lignes et 3 colonnes) numrots de 1 6.

In [64]:
x=np.linspace(0,np.pi,31)
plt.suptitle("Diffrents graphes") # le titre gnral

plt.subplot(2,2,1) # une grille 2x2 - premier graphique
plt.axis((0,np.pi,0,1))
y=np.sin(x)
plt.title('sinus')
plt.xticks([0,1,2,3])
plt.plot(x,y,"b-")

plt.subplot(2,2,2) # second graphique
plt.axis((0,np.pi,0,1))
plt.title('sinus points')
plt.plot(x,y,"go")

plt.subplot(2,2,3) # troisime graphique
y=np.cos(x)
plt.axis((0,np.pi,-1,1))
plt.title('cosinus')
plt.plot(x,y,"b-")

plt.subplot(2,2,4) # et le dernier
plt.axis((0,np.pi,-1,1))
plt.title('cosinus points')
plt.plot(x,y,"go")

plt.show() # on affiche le tout

Affichage d’une matrice¶

import numpy as np
import matplotlib.pyplot as plt
def f(x,y):
    return (2*x-y**2+y)*np.exp(-x**2-y**2)
x=np.linspace(-2,2,100) # discrtisation de [-2,2]
y=np.linspace(-2,2,100) # idem
X,Y = np.meshgrid(x,y) # cration de la grille complte
Z=f(X,Y) # on value f sur la grille
plt.matshow(Z,cmap='hot') # image avec les valeurs , style de couleurs 'hot'
plt.show()
